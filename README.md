# WorkFlow

## CLONE THE REPOSITORY

Clone the project from GitHub:

```sh
# 1. Clone the repository
git clone https://github.com/akhilejoshy/flow-craft-maker.git

# 2. Navigate into the project directory
cd flow-craft-maker

# 3. Install dependencies
npm install

# 4. Create screenshots folder if it doesn't exist
mkdir screenshots

```

**RUN FOR DEVELOPMENT / TESTING**
```sh
#Start the development server:
npm run start

```

**BUILD THE APPLICATION**
```sh
#Build the production-ready applcation:
npm run dist

# After the build completes, the application will be available inside the:
# dist/
# folder.

```
